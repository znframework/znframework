Resmi Boyutlandırmak

image::thumb( string $dosya_yolu , array $ayarlar )

Resmin genişlik ve yükseklik değerlerini ayarlamak yeniden boyutlandırmak gibi işlemleri yapmak için kullanılır. 2 Parametresi vardır parametresi vardır. Dosya Yolu, Ayarlar. Yöntem dönüş değeri olarak yeni oluşturulan resmin URL'sinin verir. Herhangi bir şekilde orjinal resim üzerinde değişiklik yapmaz ve oluşturduğu yeni resmi çağrılan resmin bulunduğu dizine thumbs/ dizini olşuturup içine ekler.

Image::thumb() yöntemi ile neler yapabilirsiniz?
1. Herhangi bir resmi kırpabilirsiniz.
2. Herhangi bir resmi yeniden boyutlandırabilirsiniz.
3. Herhangi bir resmi orantılı olarak yeniden ölçülendirebilirsiniz..

ParametreAnlamları
1. Parametre = Dosya YoluBoyutlandırılıcak resmin yolunu ifade eder.
2. Parametre = AyarlarResmin boyutlandırılması adına bir dizi ayar içerir.
2. Ayarlar Parametresi İçerisinde Kullanılan DeğerlerAnlamları
x => değerResmi yatay düzlemde kaçıncı pixelden kırpmaya başlayacağını ifade eder.
y => değerResmi dikey düzlemde kaçıncı pixelden kırpmaya başlayacağı ifade eder.
width => değerResmin kırpma genişliğini belirlemek için kullanılır.
height => değerResmin kırpma yüksekliğini belirlemek için kullanılır.
rewidth => değerResmin yeni genişlik değer ayarlanır.
reheight => değerResmin yeni yükseklik değer ayarlanır.
prowidth => değerEğer genişlik fazla ise bu ayara göre resmin yükseklik değeri otomatik olarak orantılı ayarlanır.
proheight => değerEğer yükseklik fazla ise bu ayara göre resmin genişlik değeri otomatik olarak orantılı ayarlanır.
quality => değerResmin kalitesini ayarlamak için kullanılır.

Resim/ dizini içerisinde 1500x544 ölçülerinde ornek.jpg adlı bir resim dosyası olduğunu varsayalım.

Aşağıda bir resmin istenilen yerden kırpılması gösteren örnek bir kod yer almaktadır.

<?php
class ResimUygulamasi
{
	function index()
    	{
    		import::library('Image');
            	$ayarlar = array(
            		'x' => 100,		 // Resmi yatay düzlemde 100. pixelden itibaren kırp
                    	'y' => 100,		 // Resmi dikey düzlemde 100. pixelden itibaren kırp
                    	'width' => 500,		 // x değerinden itibaren 500 pixel kırp
                    	'height' => 250		 // y değerinden itibaren 250 pixel kırp
            	);
        	$resim = image::thumb('Resim/ornek.jpg', $ayarlar);
            	echo $resim;  // http://www.ornek.com/Resim/thumbs/ornek-100x100px-500x250size.jpg
                echo '<img src="'.$resim.'">';  // Yeni resim görüntülenecektir.
    	}
}

Aşağıda bir resmin yeniden boyutlandırılmasını gösteren örnek bir kod yer almaktadır.

<?php
class ResimUygulamasi
{
	function index()
    	{
    		import::library('Image');
            	$ayarlar = array(
                    	'rewidth' => 500,		 // Resmin yeni genişliği 500 pixel
                    	'reheight' => 250		 // Resmin yeni yüksekliği 250 pixel
            	);
        	$resim = image::thumb('Resim/ornek.jpg', $ayarlar);
            	echo $resim;  // http://www.ornek.com/Resim/thumbs/ornek-0x0px-500x250size.jpg
                echo '<img src="'.$resim.'">';  // Yeni resim görüntülenecektir.
    	}
}

Aşağıda bir resmin orantılı olarak yeniden ölçülendirilmesini gösteren örnek bir kod yer almaktadır.

<?php
class ResimUygulamasi
{
	function index()
    	{
    		import::library('Image');
            	$ayarlar = array(
                    	'prowidth' => 500,		 // Resmin yeni genişliği 500 pixel olarak ayarlanırken yükseliğini bu miktara göre oranlar.
            	);
        	$resim = image::thumb('Resim/ornek.jpg', $ayarlar);
            	echo $resim;  // http://www.ornek.com/Resim/thumbs/ornek-0x0px-500x181size.jpg
                echo '<img src="'.$resim.'">';  // Yeni resim görüntülenecektir.
    	}
}

Resmin Orantılı Genişlik Yükseklik Değerini Almak

image::size( string $dosya_yolu , [ numeric $genislik ] , [ numeric $yukseklik ] )

Bir resmin genişliğine göre yüksekliğinin belirlenmesini sağlar. 3 parametresi vardır. Dosya Yolu, Genişlik, Yükseklik.

ParametreAnlamları
1. Parametre = Dosya YoluBoyutlandırılıcak resmin yolunu ifade eder.
2. Parametre = [Genişlik]Resmin belirtilen genişlik değerine göre yükseliğini orantılar.
3. Parametre = [Yükseklik]Resmin belirtilen yükseklik değerine göre genişliğini orantılar.

Aşağıda genişlik değerine göre yüksekliği oranlanan örnek bir kod yer almaktadır.

$olcu = image::size('Resim/ornek.jpg', 200);
echo $olcu->width;  // Çıktı: 200
echo $olcu->height;  // Çıktı: 73

Aşağıda yükseklik değerine göre genişliği oranlanan örnek bir kod yer almaktadır.

$olcu = image::size('Resim/ornek.jpg', 0, 200);
echo $olcu->width;  // Çıktı: 551
echo $olcu->height;  // Çıktı: 200

Resmin gerçek ölçüleri 1500x544 piksel varsayılmıştır.
