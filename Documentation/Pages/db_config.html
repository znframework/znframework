Veritabanı Ayarları

Veritabanı ayarlar Config/Database.php dosyasında yer almaktadır. Dosyada aşağıdaki ayarlar yer almaktadır.

Config/Database.php 
$config['Database']['driver'] 	= "mysqli";
$config['Database']['host'] 	= "localhost";
$config['Database']['user'] 	= "root";
$config['Database']['password']	= "";
$config['Database']['database'] = "test";
$config['Database']['dsn'] 	= '';
$config['Database']['server'] 	= '';
$config['Database']['port'] 	= '';
$config['Database']['appname'] 	= '';
$config['Database']['service'] 	= '';
$config['Database']['protocol'] = '';
$config['Database']['role'] 	= '';
$config['Database']['pconnect'] = false; 
$config['Database']['encode'] 	= false;
$config['Database']['prefix'] 	= "";
$config['Database']['charset'] 	= "utf8";
$config['Database']['collation'] = "utf8_general_ci";
$config['Database']['different_connection'] = array();

Bu ayarların ne işe yaradıkları aşağıdaki tabloda yer almaktadır.

AyarlarDeğişken TürüKullanımı
driverstring = mysqliKullanılacak veritabanı sürücüsü.
hoststring = localhostVeritabanı bağlantısı sağlanacak host ismi.
userstring = rootVeritabanı kullanıcı adı.
passwordstring = emptyVeritabanı kullanıcı şifresi.
databasestring = emptyBağlantı sağlanacak veritabanı ismi.
dsnstring = emptyBazı veritabanları için bağlantı sağlanacak dsn adresi.
serverstring = emptyBazı veritabanları için bağlantı sağlanacak server ismi.
portstring / numeric = emptyBazı veritabanları için bağlantı sağlanacak port bilgisi.
appnamestring = emptyUygulama ismi gerektiren bazı veritabanları için.
servicestring = emptyServis bilgisi gerektiren bazı veritabanları için.
protocolstring = emptyPortokol bilgisi gerektiren bazı veritabanları için.
rolestring = emptyPDO desteki Firebird veritabanı sürücüsü için.
pconnectboolean = falsePconnect desteği olan bazı veritabanlarının bu yöntem üzerinden bağlantı sağlamasını sağlamak için.
encodeboolean = falseŞifreleme desteği sağlayan veritabanları için.
prefixstring = emptyTablo öneki.
charsetstring = utf8Veritabanı karakter seti.
collationstring = utf8_general_ciVeritabanı karakter grubu.
different_connectionarray = emptyBu ayar birden fazla farklı veritabanına aynı anda bağlantı sağlamak için kullanılır. Parametre olarak ayar kümesi adı ve ayar kümesinin tuttuğu ayarlar dizisi girilir. Bu ayarlar oluşturulduktan sonra ayar kümesi adı ve different_connection() yöntemi ile farklı bağlantılar sağlanmış olur.
