Rounder Aracını Dahil Etmek

import::tool('Rounder')

Yuvarlama Aracı

rounder( numeric $sayi , [ numeric $ondalikli_basamak_sayisi = 0 ] ,[ string $tip = 'average' ] )

Ondalıklı ifadeleri yuvarlamak için kullanılır.

NoParametrelerAnlamları
1string/int Yuvarlanacak SayıYuvarlama işlemine tabi tutulacak sayı.
2[ string/int Ondalık = 0 ]Ondalıklı sayılarda virgülden sonra kaç basamak olacağı.
3[ string tip = "average" ]Yuvarlama tipi. 3 değeri vardır: average, up, down.
Tip Parametresinin Alabileceği Değerler
averageSayının ondalıklı kısmı 5 ve 5 ten büyükse bir üst sayıya altında ise bir alt sayıya yuvarlar
upBir üst sayıya yuvarlar
downBir alt sayıya yuvarlar

echo rounder(10.4); /* Çıktı: 10 */
echo rounder(10.6); /* Çıktı: 11 */
echo rounder(10.6, 0, 'down'); /* Çıktı: 10 */
echo rounder(10.6, 0, 'up'); /* Çıktı: 11 */
echo rounder(10.65487, 2); /* Çıktı: 10.65 */
echo rounder(10.65487, 3, 'down'); /* Çıktı: 10.654 */
echo rounder(10.65487, 3, 'up'); /* Çıktı: 10.655 */
