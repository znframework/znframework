Encode Kütüphanesini Dahil Etmek

import::library('Encode');

Şifre Oluşturma Yöntemi

encode::create( [ numeric $karakter_sayisi = 6 ] )

Rasgele şifre oluşturmak için kullanılır.

ParametreAnlamları
1. Parametre = Karakter SayısıŞifrenin kaç karakterli olacağını belirtir.

echo encode::create(6); // Çıktı: Rasgele 6 karakterli bir şifre üretir.

Altın Şifreleme Yöntemi

encode::golden( string $sifrelenecek_veri , string $sifre_eki )

2 Parametre kullanır.

ParametreAnlamları
1. Parametre = Şifrelenecek VeriŞifrelenecek veri girilir.
2. Parametre = Şifre EkiVeriye eklenip şifrelenir.

echo encode::golden('veriyi şifrele','£>#$^'); // Çıktı: 32 bitlik bir şifre üretecektir.

Böylece aynı veriyi farklı şekilde şifreleyebilirsiniz.

Süper Şifreleme Yöntemi

encode::super( string $sifrelenecek_veri )

Bu yöntemin golden() şifrelemeden farklı şifre eki bir dosyada tutulmaktadır. Böylece her bir projede kullanılan şifre diğer projelerde kullanılan şifrelerden farklı olacaktır. Ayrıca bu şifreleme yöntemi host adına görede şifrelenir yani bir hosttaki şifre diğer hosttaki şifreden farklı olacaktır.

Config/Encode.php dosyasında project_key ibaresine bir şifre girmeniz gerekmektedir yani herhangi bir proje anahtarı girmeniz gerekmektedir. Böyle bir uygulamanın tek amacı kullanıcıların şifre güvenliğini sağlamaktır.

ParametreAnlamları
1. Parametre = Şifrelenecek VeriŞifrelenecek veri girilir.

echo encode::super('veriyi şifrele'); // Çıktı: 32 bitlik bir şifre üretecektir.

Farklı Türlerde Şifreleme Yapmak

encode::type( string $metin , [ string $tipi = 'md5' ] )

Şifreleme için kullanılır.

NoParametrelerAnlamları
1string MetinŞifrelenecek metin.
2[ string Tip = "md5" ]Şifreleme türü. Varsayılan: md5
Şifreleme Tipleri
1md5
2sha1
3Hash algoritmalarından herhangi biri( md2 , md4 , sha256 , ripemd128 , crc32 ..)

echo encode::type('Metin'); /* Çıktı: 82f86b3ab8d192591fecc24a4db76a6b */
echo encode::type('Metin' , 'sha1'); /* Çıktı: 2c8a418f0b90694827c9d7735c041c5709fc01f9 */
