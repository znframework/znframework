Regex Kütüphanesini Dahil Etmek

import::library('Regex');

Metinsel Veriler İçinde Eşleşme Sağlamak

reg::match( string $desen , string $metin , [ string $ek_karakter ] , [ string $kapsayici = '/' ] )

PHP preg_match() ve preg_match_all() fonksiyonlarının düzenlenmiş ve belli bir standarta oturtulmuş halidir. 4 parametresi vardır. Desen, Metin, Ek Karakter /gi gibi, Kapsayıcı //

ParametreAnlamları
1. Parametre = DesenEşleşme sağlanacak desen
2. Parametre = MetinEşleşme aranacak metin.
3. Parametre = [Ek Karakter]Desen sonuna konulan ayar karakterleri /g, /i gibi.
4. Parametre = [Kapsayıcı = '/']Desenin başına ve sonuna konulan karakter /desenler/ gibi.

Düzenli ifadelerdeki desen parametrelerinin ZN Regex sınıfında ne gibi değişikliğe uğradığı aşağıdaki tabloda gösterilmiştir.

Dönüştürülmüş ParametrelerDüzenli İfadelerdeki KarşılığıAnlamları
<non-numeric>\DSayısal karakter olmayan
<numeric>\dSayısal karakter olan
<schar>\WÖzel karakter olan sayılar, harfler ve alt tire dışında kalan karakterler.
<non-schar>\wÖzel karakter olmayan sayılar, harfler ve alt tire karakterleri.
<char>.Herhangi bir karakteri ifade eder.
<non-space>\SBoşluk olmayan karakteri ifade eder.
<space>\sBoşluk olan karakteri ifade eder.
<starting>^Başlangıcı ifade eder.
<ending>$Bitişi ifade eder.
<repeat-z>*Kendinden önceki karakterin sıfır veya daha fazla tekrar etmesini ifade eder.
<repeat>+Kendinden önceki karakterin bir veya daha fazla tekrar etmesini ifade eder.
<whether>?Kendinden önceki karakterin olsa da olur olmasa da olurunu ifade eder.
<or>|Veya bağlacını ifade eder.
<perline-r>\rWindows için satır başını ifade eder.
<perline>\nSatır başını ifade eder.
<tab>\tTab boşluğunu ifade eder.
<esc>\eEscape karakterini ifade eder.
<hex>\xOn altılık sayı değerini ifade eder.
Parantez Grupları
reg::group()( )Parantez içerisine yazılan ifadeleri gruplamaya yarar.
reg::recount(){ }Yazılan sayı kadar tekrarı ifade eder.
reg::to()[ ]Belli bir aralığı ifade eder.
Ayar Karakterleri
<insens>iKüçük büyük harf duyarlılığını kaldırır.
<inspace>xBoşlukları dikkate almaz.
<generic>gKapsamlı tarama yapar.
<multiline>m^ ve $ karakterlerinin her satırın başını ve sonunu ifade etmesini sağlar.
<each>sHer karakteri ifade eden nokta karakterinin satır başlarınıda ifade etmesini sağlar.

ÖNEMLİ
Bu değişiklikler Config/Regex.php dosyasında yer almaktadır.

NOT
Desenler yazılırken bazı özel karakterleri normal karakter gibi kullanmak için başlarına konulan taksim(\) işaretinin kullanımına gerek yoktur. İstisnası sadece parantez grupları ve (-) tire'dir..

Şimdi örnek kodlar üzerinde sınıfı anlamaya çalışalım.

import::library('Regex');

$metin = 'http://www.zntr.net';

$desen = '<char><repeat-z>://w<repeat>.<char><repeat-z>.net';  //  '.*:\/\/w+\..*\.net'; 

$sonuc = reg::match($desen, $metin);

var_dump($sonuc);  // Çıktı: array (size=1) 0 => string 'http://www.zntr.net' (length=19)
Dikkat edilirse yukarıda kullanılan / veya . gibi özel karakterlerin başına \ işareti getirilmemiştir buna rağmen normal bir karakter gibi değerlendirilmiştir.

Şimdi de e-posta kontrolünü sağlayan desenin regex sınıfında nasıl yazıldığını görelim.

import::library('Regex');

$metin = 'bilgi@zntr.net';

/*REG MATCH*/

 // klasik desen = '/^([a-z0-9\+_\-]+)(\.[a-z0-9\+_\-]+)*@([a-z0-9\-]+\.)+[a-z]{2,6}$/ix' 
$desen1 = '<starting>([a-z0-9+_-]<repeat>)(.[a-z0-9+_-]<repeat>)<repeat-z>@([a-z0-9]<repeat>.)<repeat>[a-z]{2,6}<ending>';

$sonuc1 = reg::match($desen1, $metin, '<insens><inspace>');

var_dump($sonuc1); 
/* Çıktı: array (size=4)
  0 => string 'bilgi@zntr.net' (length=14)
  1 => string 'bilgi' (length=5)
  2 => string '' (length=0)
  3 => string 'zntr.' (length=5) 
*/

/*REG MATCH_ALL*/

$desen2 = '<starting>'.
            reg::group(reg::to('a-z0-9+_-').'<repeat>').
            reg::group('.'.reg::to('a-z0-9+_-').'<repeat>').'<repeat-z>@'.
            reg::group(reg::to('a-z0-9').'<repeat>.').'<repeat>'.
            reg::to('a-z').reg::recount('2,6').
          '<ending>';

$sonuc2 = reg::match_all($desen2, $metin, '<insens><inspace>');

var_dump($sonuc2); 
/* Çıktı: array (size=4)
  0 => 
    array (size=1)
      0 => string '2bilgi@zntr.net' (length=15)
  1 => 
    array (size=1)
      0 => string '2bilgi' (length=6)
  2 => 
    array (size=1)
      0 => string '' (length=0)
  3 => 
    array (size=1)
      0 => string 'zntr.' (length=5)
*/
Yukaridaki kodda iki farklı desen yazılmıştır siz hangisi kolayınıza geliyorsa o tipi kullanabilirsiniz. Pratik ve akılda kalıcı olmadığını düşünüyorsanız klasik preg_match() yöntemini kullabilirsiniz.

Eşleşen Veriler Değiştirmek

reg::replace( string $degisecek_veri_deseni , string $yeni_karakter , string $metin , [ string $ek_karakter ] , [ string $kapsayici = '/' ] )

PHP preg_replace() fonksiyonunun düzenlenmiş ve belli bir standarta oturtulmuş halidir. 5 parametresi vardır. Değişecek Veri Desene , Yeni Karakter, Metin, Ek Karakter /gi gibi, Kapsayıcı //

ParametreAnlamları
1. Parametre = DesenEşleşme sağlanacak desen
2. Parametre = Yeni KarakterEski karakterin yerini alacak yeni karakter.
3. Parametre = MetinEşleşme aranacak metin.
4. Parametre = [Ek Karakter]Desen sonuna konulan ayar karakterleri /g, /i gibi.
5. Parametre = [Kapsayıcı = '/']Desenin başına ve sonuna konulan karakter /desenler/ gibi.

Aşağıdaki örnek kodda reg::replace() kullanımına yer verilmiştir.

import::library('Regex');

$metin = 'http://www.zntr.net';

$degisecek_veri_deseni = 'www.'; // kasik desen = /www\./

$sonuc = reg::replace($degisecek_veri_deseni, '', $metin);

echo $sonuc;  // Çıktı: http://zntr.net

ÖNEMLİ
Düzenli ifadelerdeki karakter değişiklikleri Config/Regex.php dosyasında yer almaktadır.
